### 安装主题
``` sh
yarn add @qcyblm/vuepress-theme-vpx --registry=https://npm.pkg.github.com/
# or npm i @qcyblm/vuepress-theme-vpx --registry=https://npm.pkg.github.com/
```
### 引用主题
``` js
// .vuepress/config.js

module.exports = {
  ...
  theme: '@qcyblm/vpx',
  ...
}
```
::: warning
请确保你的 Node.js 版本 >= 8.6。
:::
